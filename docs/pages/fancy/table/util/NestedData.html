package fancy.table.util

class NestedData

Static methods

staticgenerateFolds (data:Array<RowData>, start:Int = 0):Tuple2<Int, Array<Tuple2<Int, Int>>>

Given an array of (nested) row data, this function returns a tuple with fold information. The left side of the resulting tuple is the number of rows processed. The right side of the tuple is an array of fold tuples. The left side of the fold tuple is a header row index, and the right side is a count of how many rows should be folded under it.

  // generate folds, then iterate over the array of fold tuples
  generateFolds(data).right.reduce(function (fold) {
    return table.createFold(fold.left, fold.right)
  }, table); // assumes you already have a table to start with

staticrectangularize (data:Array<RowData>):Array<Array<String>>

Converts a nested RowData structure into a flat array of array of strings, which can be used to set complete table data from scratch.
